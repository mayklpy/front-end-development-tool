# Front-end Development Tool

Front-end development tool using Node.js, Babel, Webpack, PostCSS and Sass

## Setup

```shell
$ npm install
```

## Development mode

```shell
$ npm run dev
```

Development mode is running at http://<span></span>localhost:3000/

## Production Mode

```shell
$ npm run build
```

## Preview the build

```shell
$ npm run serve
```
